# dio-lab-open-source

Repositório de exercícios e projetos abertos para aprendizado e contribuição.

## Descrição
Um espaço para praticar, aprender e contribuir com projetos de código aberto — ideal para iniciantes e para quem quer colaborar.

## Como usar
1. Clone o repositório:
```bash
git clone https://github.com/digitalinnovationone/dio-lab-open-source.git
```
2. Entre na pasta do projeto:
```bash
cd dio-lab-open-source
```
3. Siga os READMEs das pastas específicas (por exemplo, `community/`) para instruções de contribuição.

## Contribuindo
- Faça um fork
- Crie uma branch: `git checkout -b minha-contribuicao`
- Abra um PR com uma descrição clara do que foi alterado
