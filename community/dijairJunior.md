# Projetos Pessoais

### Gerador de Números da Mega-Sena
#### Descrição
Este projeto consiste em um gerador de números da Mega-Sena em Python, que utiliza dados históricos fornecidos por um arquivo CSV para gerar números potenciais com base nas análises das colunas do CSV.

#### Funcionalidades Principais:
- Leitura de Dados do Arquivo CSV: O programa lê os dados do arquivo CSV fornecido, ignorando o cabeçalho e armazenando os números por coluna.
- Análise de Números Mais Frequentes: Analisa os números mais frequentes em cada coluna do CSV.
- Geração de Números Aleatórios: Com base na análise das colunas, gera números aleatórios garantindo que não sejam iguais aos números mais frequentes de cada coluna e aos números gerados anteriormente.
- Iteração com o Usuário: Oferece uma interface simples para o usuário, permitindo gerar números da Mega-Sena e decidir se deseja gerar mais números.

#### Tecnologias utilizadas.
- Python
- Bibliotecas: CSV, Random, Collections

### Implementação Simples de IA

#### Descrição
Este projeto consiste em uma implementação básica de um assistente virtual em Python, denominado "Athena". A finalidade do assistente é auxiliar os usuários em diversas tarefas por meio da interação por voz, fornecendo informações, realizando pesquisas na internet e executando comandos de forma simples e intuitiva.
