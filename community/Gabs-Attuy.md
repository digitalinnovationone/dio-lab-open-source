<h1 align="center">Gabriel Santos Attuy</h1>

<h3 align="center">
Backend Software Developer • C# • .NET • Java • Spring Boot
</h3>

Sou Analista e Desenvolvedor de Software com foco no desenvolvimento de aplicações backend utilizando C#, .NET, Java e Spring Boot.

Tenho experiência no desenvolvimento de aplicações backend utilizando os ecossistemas .NET e Spring, criando APIs REST, implementando autenticação JWT, persistência com bancos relacionais e NoSQL, conteinerização com Docker e integração com serviços em nuvem como AWS e Azure.

Gosto de transformar problemas de negócio em soluções bem estruturadas, aplicando princípios como Clean Code, SOLID, testes automatizados e boas práticas de arquitetura para desenvolver aplicações escaláveis, seguras e de fácil manutenção.

Atualmente continuo aprofundando meus conhecimentos em arquitetura de software, microsserviços e computação em nuvem, buscando evoluir continuamente como desenvolvedor backend.
