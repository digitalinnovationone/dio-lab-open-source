# Guia Rápido de Contribuição


- **Fork:** Clique em "Fork" no GitHub para criar sua cópia pessoal do projeto.

- **Clone:** No terminal, baixe sua cópia: `git clone URL_DO_SEU_FORK`.
- **Branch:** Crie uma área de trabalho: `git checkout -b feat/community/SEU_USUARIO`.
-  **Criar:** Na pasta `community`, crie o arquivo `SEU_USUARIO.md` com seu perfil.
- **Add:** Prepare o arquivo para salvar: `git add community/SEU_USUARIO.md`.
- **Commit:** Grave a mudança: `git commit -m "feat: add SEU_USUARIO profile"`.
- **Push:** Envie para o seu GitHub: `git push origin feat/community/SEU_USUARIO`.
- **Pull Request (PR):** No site do GitHub, clique em "New Pull Request" e envie.
- **Dica:** Use `feat:` para novas inclusões e `fix:` para correções de erros.
- **Atenção:** O nome do arquivo .md deve ser idêntico ao seu user do GitHub.
- **Validação:** Após o PR, aguarde a automação aprovar sua contribuição.
- **Comunidade:** Explore a pasta `utils` para adicionar badges e cards ao seu perfil.
