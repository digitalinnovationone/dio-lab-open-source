### Guia de Contribuição

## Convenção de Commits 

| Tipo de Commit | Descrição                                                            | Exemplo
| ---------------|----------------------------------------------------------------------|-----------
| `feat`         | Adiciona uma nova funcionalidade ao projeto.                         | `feat: add user login feature`
| `fix`          | Corrige um bug ou problema no projeto.                               | `fix: resolve crash on payment gateway`
| `docs`         | Altera a documentação do projeto.                                    | `docs: update API usage section in README`
| `style`        | Realiza mudanças na aparência, sem alterar a funcionalidade.         | `style: fix indentation in main.css`
| `refactor`     | Realiza mudanças no código que não alteram a funcionalidade.         | `refactor: optimize database query logic`
| `test`         | Adiciona ou modifica testes no projeto.                              | `test: add integration tests for OrderController`
| `perf`         | Melhora o desempenho de uma parte do sistema.                       | `perf: optimize image loading for homepage`
| `chore`        | Realiza tarefas rotineiras ou de manutenção.                         | `chore: update npm dependencies`
| `build`        | Realiza mudanças no sistema de build ou dependências.               | `build: configure webpack for production mode`
| `ci`           | Modifica configurações de integração contínua.                      | `ci: fix deployment pipeline error`
| `revert`       | Reverte um commit anterior.                                          | `revert: revert "add new payment method"`
| `env`          | Altera configurações de ambiente.                                   | `env: add .env.example file`
| `hotfix`       | Realiza uma correção emergencial em produção.                        | `hotfix: fix critical login bug`
| `deps`         | Atualiza ou altera dependências do projeto.                         | `deps: upgrade axios to v1.2.3`
| `config`       | Modifica arquivos de configuração.                                  | `config: update ESLint rules`
| `security`     | Implementa melhorias de segurança.                                  | `security: fix XSS vulnerability in user input`
| `ux`           | Melhora a experiência do usuário.                                   | `ux: adjust button size for better mobile usability`
| `ui`           | Altera elementos da interface do usuário.                          | `ui: update header layout for responsiveness`
| `i18n`         | Adiciona ou modifica internacionalização.                           | `i18n: add French translations`
| `l10n`         | Ajusta localizações específicas.                                    | `l10n: update currency format for EU countries`
| `compat`       | Garante compatibilidade com versões ou sistemas.                    | `compat: add polyfill for older browsers`
| `deprecate`    | Marca funcionalidade como obsoleta.                                 | `deprecate: deprecate legacy user API`
| `remove`       | Remove código ou arquivos desnecessários.                           | `remove: delete unused test files`
| `analytics`    | Adiciona ou modifica análises ou rastreamentos.                     | `analytics: track user clicks on signup button`
| `accessibility`| Implementa melhorias de acessibilidade (A11y).                      | `accessibility: add ARIA labels to navigation menu`
