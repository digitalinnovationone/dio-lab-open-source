## Sobre mim

Em primeiro lugar, bem-vindo ao meu perfil, espero que possamos compartilhar conhecimentos e experiências.

Sou formado em Gestão em Tecnologia da Informação e atualmente atuo como desenvolvedor para uma empresa de software que atende diversas empresas, dentre elas, uma grande instituição financeira onde estou alocado, e nesta a linguagem de programação predominante é o Java.

No frontend tenho experiência com Typescript, utilizando a seguinte tecnologia: Angular 7, HTML5, Bootstrap e CSS, já no backend minha experiência é com JAVA, APIs Rest com Spring Boot e banco de dados PostgreSQL.
