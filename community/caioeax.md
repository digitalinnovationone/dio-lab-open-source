## Projetos e Cursos

</div>

- [The Conundrum](https://github.com/STB-Games/The-Conundrum)  
  Jogo que desenvolvi em parceria com uma colega. O jogo consiste no mistério em que dois detetives contratados para investigar uma mansão no interior do Amazonas se localizam e precisam resolver uma série de puzzles e desafios para tentar desvendar a lenda local. O jogo conta com VOIP e história própria, recomenda-se jogar com celulares.

- [Curso Python](https://www.udemy.com/course/python-3-do-zero-ao-avancado/)  
  O curso "Python 3 do básico ao avançado - com projetos reais" da Udemy ensina lógica de programação, POO, banco de dados, web scraping, automação e mais, com foco em prática através de projetos reais.

- [Curso Banco de Dados (MySQL)](https://www.udemy.com/course/mysqlcompleto/)  
O curso “MySQL Completo - 123 horas, formação DBA" ensina MySQL, SQL, Azure MySQL, Modelagem de
Dados, Windows Server e Linux Ubuntu Server - do básico ao avançado com foco em exemplos e práticas que podem acontecer no dia a dia empresarial.
