# Rubem Tanaami
Trabalhei como editor de vídeo durante muito tempo. Durante o meu trabalho com video me deparei com script para que pudesse mexer com alguns modificadores. Pensei que seria dificil mas inesperadamente foi muito intuitivo para mim. Mais tarde lembrei que durante a minha infância eu programa joguinhos simples em basic. Resolve dar uma chance para algo que gostava de fazer mas que não sei porque parei de fazer, que é programar.
