# OLÁ! SEJA BEM VINDO(A)

### Meu nome é Jêdisson, sou ex-(quase) estudante de jornalismo, em um relacionamento sério com a tecnologia! Embora minha relação com esse mundo tenha tido início na infância, iniciei minha jornada na programação em 2022 e, desde então, tenho mergulhado de cabeça no mundo da tecnologia.

### E ao Longo desse caminho desenvolvi uma vasta gama de projetos utilizando algumas das seguintes ferramentas e funcionalidades:

- **-Construção de servidores web em node.js utilizando o express.**
- **-Banco de dados relacional, com PostgreSQL e, não relacional, com MongoDB.**
- **-Utilização de query builders, como o knex, para consultas em base de dados.**
- **-Autenticação utilizando o JWT (JSON Web Token) e encriptação de senhas ou credenciais de entrada sensíveis com o bcrypt.**
- **-Conceito de POO (programação orientada a objetos), com typescript.**
- **-Utilização de ORM para manipulação abstrata de dados, utilizando o typeorm.**
- **-Manipulação e integração de servidores SMTP e de armazenamento nas aplicações utilizando o AWS-SDK**
- **-Integração com APIs de terceiros utilizando clientes HTTP como o Axios.**
- **-Deploy de aplicações na vercel, cyclic e Heroku.**
- **-Versionamento de códigos no Github.**
- **-Metodologias ágeis: scrum e Kanban.**
