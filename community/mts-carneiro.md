## SOBRE MIM

Com anos de experiência na área de gerenciamento de cozinha, decidi fazer uma transição de carreira para a programação, algo que ate então era um hobby.
Decidido, dediquei os meus esforços para a programação, onde aprendi a desenvolver aplicações responsivas em React Js, utilizando JavaScript e Type Script, e aplicando o designer através CSS.
Em seguida estudei sobre Back-end, onde aprendi a desenvolver APIs com o Django e JavaScript através do Express, armazenando os dados manipulados em um banco de dados PostgresSQL.
Atualmente estudo na área do Front-end o Angular, e no Back-end estudo o Nest.js, com o intuito de ampliar meus conhecimentos e melhorar minhas habilidades.
