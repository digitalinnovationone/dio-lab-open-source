## 📖 Sobre o Projeto

Bem-vindo(a)! Este é um projeto educacional desenvolvido para o curso **"Contribuindo em um Projeto Open Source no GitHub"** da [Digital Innovation One](https://www.dio.me/).

O objetivo principal é **ensinar iniciantes a contribuir com projetos open source**, praticando conceitos importantes como:

- ✨ Criar e editar arquivos README
- 🔀 Fazer um fork de repositórios
- 📝 Criar pull requests
- 🤝 Colaborar com a comunidade de desenvolvedores

> **💡 Importante:** Este projeto usa Markdown para documentação. Markdown é uma linguagem de marcação simples usada para formatar textos (criar títulos, listas, links, etc.). É perfeito para documentação de projetos!

## 📁 Estrutura do Projeto

Entenda como os arquivos estão organizados:

```
docs/
├── assets/              # Pasta com recursos do projeto
│   ├── css/            # Arquivos de estilo
│   │   └── styles.css  # Folha de estilos principal
│   └── js/             # Arquivos JavaScript
│       └── scripts.js  # Código JavaScript principal
├── favicon.ico         # Ícone que aparece na aba do navegador
├── index.html          # Página principal do site
└── README.md           # Este arquivo (documentação do projeto)
```

### 📝 O que cada arquivo faz?

- **index.html**: A página principal em HTML que estrutura o conteúdo
- **styles.css**: Define as cores, fontes e layout da página
- **scripts.js**: Adiciona interatividade e funcionalidades à página
- **README.md**: Documentação explicando o projeto (você está lendo ele agora!)

## 🚀 Como Usar Este Projeto

### Pré-requisitos

Antes de começar, você vai precisar ter instalado:

- Um navegador web (Chrome, Firefox, Edge, etc.)
- [Git](https://git-scm.com/) instalado no seu computador
- Uma conta no [GitHub](https://github.com/)
- Um editor de código (recomendamos o [VS Code](https://code.visualstudio.com/))

### Passo a Passo

1️⃣ **Faça um Fork do Projeto**
   - Clique no botão "Fork" no topo da página do repositório
   - Isso criará uma cópia do projeto na sua conta

2️⃣ **Clone o Repositório**
   ```bash
   git clone https://github.com/SEU-USUARIO/dio-lab-open-source.git
   ```

3️⃣ **Entre na Pasta do Projeto**
   ```bash
   cd dio-lab-open-source
   ```

4️⃣ **Abra o Projeto**
   - Abra a pasta `docs` no seu editor de código
   - Clique duas vezes no arquivo `index.html` para visualizar no navegador

5️⃣ **Faça suas Alterações**
   - Edite os arquivos conforme necessário
   - Salve as alterações

6️⃣ **Commit e Push**
   ```bash
   git add .
   git commit -m "Descrição das suas alterações"
   git push origin main
   ```

7️⃣ **Crie um Pull Request**
   - Vá até o repositório original
   - Clique em "Pull Request" → "New Pull Request"
   - Descreva suas alterações e envie!

## 🤝 Como Contribuir

Contribuições são sempre bem-vindas! Aqui estão algumas formas de ajudar:

1. 🐛 Reportar bugs ou problemas
2. 💡 Sugerir novas funcionalidades
3. 📝 Melhorar a documentação
4. ✨ Enviar código para corrigir problemas

### Dicas para Iniciantes

- Comece com mudanças pequenas (corrigir um erro de digitação, adicionar uma linha na README)
- Leia as contribuições de outros para aprender
- Não tenha medo de perguntar na comunidade!
- Teste suas alterações antes de enviar
