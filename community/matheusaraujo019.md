Este repositório foi criado para documentar minha jornada e prática no ecossistema de código aberto. O objetivo principal foi entender o fluxo de trabalho colaborativo utilizando Git e GitHub.

* **Link da Pull Request:** [Cole o link da sua PR aqui]

## 🧠 Aprendizados
Durante o desafio, pude praticar:
1. **Fork & Clone:** Criação de uma cópia local para desenvolvimento.
2. **Feature Branch:** Organização do trabalho em branches específicas.
3. **Pull Request (PR):** Submissão de código para revisão e integração.
4. **Code Review:** Entendimento das boas práticas exigidas pela comunidade.
