## Resumo Profissional:
Com mais de 15 anos de sólida experiência na indústria da construção civil, estou empenhado em combinar meu vasto conhecimento com uma paixão recente: o desenvolvimento mobile nativo Android com Kotlin. Minha trajetória inclui a conclusão do curso de Análise de Desenvolvimento de Sistemas, além da criação de três aplicativos nativos ("matches-simulator," "soccer-news" e "business-card") que demonstram minha proficiência em desenvolvimento móvel, programação Android, Git e GitHub. Minha habilidade de transpor habilidades técnicas complexas para soluções práticas é enriquecida por uma base sólida em engenharia.

#### Projetos Desenvolvidos:

#### matches-simulator-app:
- Um aplicativo de simulação de resultados de partidas de futebol. Utilizei o GitHub Pages para simular uma API Rest e consumi-a através do Retrofit. Gerenciei imagens com o Glide.

#### soccer-news-app:
- Criei um aplicativo de notícias de futebol feminino, permitindo aos usuários favoritar e compartilhar notícias. Utilizei o GitHub Pages para simular a API Rest e o Retrofit para consumir os dados. Gerenciei imagens com o Picasso e utilizei o Room para salvar notícias favoritas.

#### business-card-app:
- Desenvolvi um aplicativo para criação e compartilhamento de cartões de visitas. Usei o Room para a persistência dos dados.
