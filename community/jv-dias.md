### 🔍 Sobre o Projeto

O **QuiosqueBI** é uma plataforma inovadora de Business Intelligence que desenvolvi do zero, combinando análise de dados com **Inteligência Artificial Generativa**. A plataforma permite que usuários façam upload de planilhas e, usando **linguagem natural**, recebam análises detalhadas e gráficos gerados dinamicamente pelo **Google Gemini AI**.

### 🛠️ Tecnologias Utilizadas

- **Backend:** .NET 8, C#, Entity Framework, Clean Architecture
- **Frontend:** Vue.js 3, TypeScript, Composition API
- **Database:** PostgreSQL
- **IA:** Google Gemini API para processamento de linguagem natural
- **Cloud:** Microsoft Azure (Static Web Apps, App Service, Azure Database)
- **DevOps:** GitHub Actions para CI/CD automatizado

### 🌐 Links do Projeto

- **🚀 Aplicação Online:** [quiosquebi.azurestaticapps.net](https://victorious-dune-05e42d21e.1.azurestaticapps.net)
- **📱 Frontend (Vue.js):** [jv-dias/QuiosqueBI](https://github.com/jv-dias/QuiosqueBI)
- **⚙️ Backend (.NET):** [jv-dias/QuiosqueBI-Backend](https://github.com/jv-dias/QuiosqueBI-Backend)

### 🎯 Principais Funcionalidades

- Upload e processamento de planilhas Excel/CSV
- Consultas em linguagem natural para análise de dados
- Geração automática de gráficos e insights
- Interface intuitiva e responsiva
- Arquitetura escalável com deploy automatizado
