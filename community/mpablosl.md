## 🚀 Minhas Skills
Minha Stack Favorita 🐍
Minha stack principal é centrada em Python, uma linguagem que me fascina pela sua versatilidade e clareza. Sou adepto do Flask para desenvolvimento web, valorizando sua simplicidade e flexibilidade. ## Projetos em Destaque 🌟

Meu primeiro projeto prático no desenvolvimento de software foi a criação de um CRUD (Create, Read, Update, Delete) de uma lista de tarefas. Esse projeto foi fundamental para consolidar meus conhecimentos em:

Implementação de APIs: Entender como construir endpoints para interagir com o banco de dados.
Manipulação de Dados: Praticar a inserção, leitura, atualização e exclusão de registros.
Integração com Banco de Dados: Conectar o Flask com o SQLite e MySQL utilizando SQLAlchemy.
Este projeto me proporcionou uma base sólida para desenvolver aplicações web mais complexas e eficientes.
