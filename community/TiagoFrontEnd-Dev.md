## 🚀 Objetivo do Desafio

- Aprender na prática o fluxo de contribuição open source;
- Criar um Profile README com informações sobre você;
- Utilizar Fork, Pull Request e Markdown de forma colaborativa;
- Reforçar boas práticas de versionamento com Git e GitHub.
