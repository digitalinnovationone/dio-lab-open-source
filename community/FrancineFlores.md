\## Como escrever arquivo utilizando linguagem Markdown



Markdown é uma linguagem de marcação leve e simples que usa sinais de texto simples para formatar documentos, tornando-os mais fáceis de ler e escrever do que linguagens como HTML. Ela é amplamente usada para formatar texto na web, como em arquivos README, fóruns, comentários de tickets e documentos. Seu principal benefício é a legibilidade e a simplicidade, que permite que documentos sejam formatados sem a necessidade de uma interface visual complexa. 

Como funciona

Sintaxe simples: Usa caracteres de pontuação como \*, # e ~ para criar formatação.

Conversão para HTML: O Markdown é convertido em HTML válido, facilitando a apresentação de conteúdo em ambientes online.

Legibilidade: Mesmo sem ser renderizado, o texto é fácil de ler e entender, pois a estrutura de formatação é visível.
