### Sobre Mim
Olá! Eu sou o Henrique, um entusiasta de 21 anos cursando Engenharia Mecânica e trilhando meu caminho no universo da Ciência de Dados e Análise de Dados.
Tenho me dedicado a explorar as possibilidades do Python, SQL, Excel e Power BI, aplicando essas ferramentas para transformar dados em informações valiosas e gerar insights.

### Objetivos
Este repositório é o meu espaço para documentar minha jornada, compartilhar projetos práticos e mostrar minha evolução na área.
Aqui, você encontrará trabalhos que envolvem:

- Python para manipulação e análise de dados

- SQL para consulta e gestão de bancos de dados

- Excel para organização, modelagem e cálculos

- Power BI para visualização de dados e criação de dashboards interativos

- Conteúdos e artigos que publico no Medium sobre Ciência de Dados e Análise de Dados

Meu objetivo é não apenas aprender, mas também contribuir de forma significativa para a comunidade, trocando experiências e boas práticas.

### Como Você Pode Contribuir
Seja você um estudante, um profissional experiente ou apenas alguém apaixonado por Ciência de Dados e Python, sua contribuição é valiosa. Fique à vontade para explorar meus projetos, fornecer feedback ou sugerir novas ideias. Vamos aprender e crescer juntos!
