Como contribuição envio um tutorial para gerar um arquivo Markdown.

Como gerar um arquivo .md:
1. Crie um novo arquivo:
No seu editor de texto, crie um novo arquivo e salve-o com a extensão .md (ex: meu_arquivo.md).
2. Digite o texto:
Comece a digitar o seu texto e utilize a sintaxe Markdown para formatar. 
3. Visualização:
Se estiver usando um editor Markdown dedicado, você poderá ver a visualização do seu texto formatado em tempo real. 
4. Conversão para HTML:
Dependendo do seu objetivo, você pode converter o arquivo .md para HTML usando ferramentas online ou plugins de edição. 


Sintaxe básica de Markdown:
Cabeçalhos: Use # para criar cabeçalhos (ex: # Título Principal, ## Subtítulo).
Negrito: Use ** para colocar texto em negrito (ex: **texto em negrito**).
Itálico: Use * para colocar texto em itálico (ex: *texto em itálico*).
Listas: Utilize * ou - para criar listas não ordenadas, e números seguidos de ponto para listas ordenadas.
Links: Use nome do link.
Imagens: Use !nome da imagem.
