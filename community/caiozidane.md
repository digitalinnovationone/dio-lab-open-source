# Caio Zidane


Sou um profissional de Desenvolvimento Web com experiência variada em várias empresas. Atuei como Frontend Web Developer na Datweb Sistemas, onde sustentei o produto legado principal usando PHP no backend e JavaScript no frontend. Anteriormente, na Aprosoja MT, desenvolvi projetos web com Laravel. Na Squadra Tecnologia, trabalhei como Desenvolvedor Júnior Java, desenvolvendo API Rest com SpringBoot e sistemas legados em Java (6,7 e 8). Como Estagiário PHP na SESP, contribuí para projetos internos com a ferramenta Liferay. Tenho habilidades em HTML, CSS, JavaScript, e estou pronto para contribuir efetivamente em projetos de desenvolvimento web.
