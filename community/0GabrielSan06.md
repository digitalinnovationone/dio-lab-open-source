# Alguns comandos básicos do git

Sou João Gabriel e essa será uma pequena lista de comandos básicos com git.


## Criando e Iniciando um repositório

#### Cria um repositório local

```http
  mkdir [NOME DA PASTA]
```


#### Acessar diterórios

```http
  cd [NOME DA PASTA]
```


#### Iniciar um repositório

```http
  git init
```


## Ligando e Clonando repositório

Após criar o repositório remoto no github, é necessário ligá-lo ao seu repositório local.

#### Ligar o repositório local ao remoto

```http
git remote add origin [URL DO REPOSITÓRIO DO GITHUB]
```


#### Clonar repositório
```http
git clone [URL DO REPOSITÓRIO DO GITHUB]
```

#### Clonar apenas uma branch do repositório

```http
git clone [URL DO REPOSITÓRIO] --branch [NOME DA BRANCH]
```
