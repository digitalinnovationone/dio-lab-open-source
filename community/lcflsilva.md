## 🚀 Projeto Principal: **Bandástico**  

📱 O **Bandástico** é um aplicativo que conecta **amantes da música**, **músicos** e **a cena independente**.  
A ideia é centralizar informações de bandas, shows e artistas em um só lugar, apoiando a cultura musical de forma interativa.
O projeto foi desenvolvido como meu trabalho de conclusão de curso, utilizando o Framework Cross Plataform Expo com React Native e Typescript, além do Firebase

🔗 Em breve disponível em: [www.bandastico.com.br](http://www.bandastico.com.br)
