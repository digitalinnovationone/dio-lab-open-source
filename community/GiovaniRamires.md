## 📌 Projetos em destaque

### 🔹 [Sistema Kanban em Java](https://github.com/GiovaniRamires/sistema-kanban-java)
Sistema de gerenciamento de tarefas estilo Kanban com persistência em **MySQL**, construído em arquitetura de 4 camadas (**Model · DAO · Service · View/Controller**).

- Herança e polimorfismo: `Bug` e `Feature` estendem a classe abstrata `Tarefa`, cada uma com sua própria regra de prioridade e prazo
- Persistência via **JDBC** com `PreparedStatement` (protegido contra SQL Injection) e *try-with-resources*
- Padrão **DAO** isolando todo acesso a dados da camada de negócio
- Registro de histórico: cada mudança de status gera um evento com timestamp
- Relatórios de backlog priorizado por severidade (Bug) ou valor de negócio (Feature)
- 🔄 **Em andamento:** migração da camada de acesso a dados e da aplicação para **Spring Boot** (branch aberta), evoluindo de JDBC puro para Spring Data JPA

`Java` `MySQL` `JDBC` `POO` `Spring Boot (em migração)`

### 🔹 [Todo List API — Spring Boot](https://github.com/GiovaniRamires/todo-list-rocketseat)
API REST desenvolvida no curso introdutório de Spring Boot da Rocketseat, com **deploy em produção**.

- CRUD completo de usuários e tarefas via endpoints REST
- Persistência com **Spring Data JPA** (banco H2)
- Containerizada com **Docker** e publicada no Render
- 🔗 Deploy: [todo-list-rocketseat-qsqw.onrender.com](https://todo-list-rocketseat-qsqw.onrender.com) *(pode levar alguns segundos para “acordar” após inatividade)*

`Java` `Spring Boot` `Spring Data JPA` `Docker` `Maven`
