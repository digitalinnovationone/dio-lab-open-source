## 🌟 Projeto em Destaque

### 📌 TCC - API de Vitrine Digital

Sistema de vitrine digital desenvolvido para pequenas lojas que desejam ingressar no mercado online de forma simples e acessível.

🔹 API completa para exibição de produtos e solicitação de pedidos
🔹 Gerenciador de pedidos estilo Kanban para controle de vendas
🔹 Painel administrativo simples para gestão de produtos e estoque
🔹 Modelagem de dados e consistência utilizando MongoDB
🔹 Aplicação de arquitetura modular bem estruturada

💡 **Diferencial:** solução pensada para ser simples, acessível e eficiente para pequenos negócios

**Tecnologias utilizadas:**

* Node.js
* Express.js
* MongoDB
* Mongoose
* Joi
