## Minha Experiência

Durante minha jornada de 6 anos como desenvolvedor, tive a oportunidade de trabalhar em diversos projetos desafiadores, abrangendo desde aplicativos web dinâmicos até soluções de back-end escaláveis. Meu foco principal tem sido o desenvolvimento de aplicações modernas e eficientes, utilizando as mais recentes tecnologias e práticas de desenvolvimento.

Tenho experiência significativa no desenvolvimento de aplicações web utilizando ReactJS, Angular e TypeScript, fornecendo interfaces intuitivas e responsivas para os usuários finais. Além disso, estou confortável trabalhando com Node.js e Fastify para construir APIs robustas e de alto desempenho, e tenho experiência em integração com bancos de dados MySQL.

Também possuo conhecimento em Java e Spring Boot, o que me permite desenvolver aplicações corporativas seguras e escaláveis quando necessário.
