# RESUMO PROFISSIONAL

Recém-formado em Análise e Desenvolvimento de Sistemas, possuo experiência intermediária em programação full-stack e automação, com participação em projetos acadêmicos e freelancers. Meus principais projetos incluem o BioEnergy, iniciativa acadêmica, e o DogBots, trabalho freelancer. Utilizando JavaScript, React, Python, Django, MongoDB, e Selenium. Também detenho conhecimentos básicos em C++, C, Java e SQL.

- BioEnergy: Refatorei o backend em JavaScript (Node.js), configurei o arquivo .env com integração da API Mailtrap e ajustei o package.json para gerenciamento de dependências. Implementei inicialmente o banco de dados MongoDB, substituído por Firebase, o que levou à descontinuação do trabalho inicial. Colaborei na análise do front-end (React e Sass) e backend, corrigindo erros e otimizando o código. Ajustei um chatbot (não implementado) para facilitar manutenção futura. O projeto está em fase de demonstração, buscando captação de investidores.

- DogBots: Desenvolvi um bot para Telegram com Python e Selenium e mais algumas outras bibliotecas PY, coletando e filtrando mensagens em tempo real, armazenadas em arquivo .txt. Apesar de planejada maior automação, o projeto foi descontinuado, resultando em um protótipo funcional, porém incompleto.
