### Projetos Finalizados 

[**Threads Text Checker**](https://github.com/joelmcfilho/threads_text_checker) : Se trata de um resumidor de textos por IA, com o intuito de reduzir textos maiores que 500 caracteres para um texto conciso menor que 500 caracteres, utilizando o modelo Gemini Flash 2.5. Além disso, o programa permite que você fracione seu texto maior em blocos de menos que 500 caracteres automaticamente.

[**Calculadora OOP**](https://github.com/joelmcfilho/calc_oop) : Uma calculadora simples feita para relembrar alguns conceitos chave de algoritmos e estruturas em Python. Interface de usuário desenvolvida a partir da biblioteca Tkinter.

[**Dashboard de Fluxo de Vendas (SAP ABAP)**](https://github.com/joelmcfilho/DESAFIO-DASHBOARD-ABAP-JR-NV-1) : Um gerador de relatório dinâmico customizado, que exibe ao usuário dados de venda, remessa e fatura de uma ordem, tudo em um mesmo relatório ALV.
