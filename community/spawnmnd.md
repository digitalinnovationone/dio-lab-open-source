# Marcelo Dias
Olá, sou estudante de Análise e Desenvolvimento de Sistemas na Estácio - Macaé-RJ. Já sou familiarizado com o mundo da progração onde fiz meus primeiros cursos na área em meados de 2017. Comecei pela lógica de programação, depois vi algumas coisas sobre HTML e CSS, Python e Javascript. Fiz tudo online através de sites como: codecademy e udemy. Aos poucos fui percebendo que o front-end é interessante mas ainda não era o ideal pra mim, pois não me causava grande entusiasmo. A virada de chave foi quando em 2021 eu fiz um curso presencial no Rio de Janeiro sobre desenvolvimento de software e aprendi mais sobre o back-end e a linguagem Java. Ali eu posso afirmar que encontrei um caminho mais agradável no meio da programação. Desenvolvi uma API utilizando Springboot que envolve conceitos como Spring Security, Spring Data JPA. Este projeto está no meu github como público pra quem quiser dar uma olhada. Hoje procuro aprofundar meus conhecimentos em Java e por isso agarrei essa oportunidade da DIO + Banco Santander neste curso sobre back-end utilizando Java. Vamos que vamos amigos!
