# Rocasouza
Sou desenvolvedor delphi, front-end (HTML, CSS, Bootstrap).
Atualmente trabalho na JF System Sistemas Comerciais, onde sou responsável pelo desenvolvimento e atualização dos relatórios do sistema em Fortes Report e o Dashboard para equipamentos Windows(Delphi VCL). Além de ser responsável também pelas migrações dos dados de outros ERPs para o nosso sistema, usando diversos bancos de dados (Postgres, MySQL, Firebird, MSSQL, ...). Desenvolvo e atualizo os nossos aplicativos Android usando Delphi Firemonkey(FMX).
