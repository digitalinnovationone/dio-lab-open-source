# Guilherme Correa

## Apresentação

Meu primeiro contato com a programação foi em 2017 utilizando a linguagem C++, confesso que no início não gostei muito, pois achava que programação era apenas matemática, array, ponteiros e coisas do tipo. No entanto, em 2019 conheci a linguagem de programação Python e isso mudou completamente meu mindset sobre programação - percebi que era útil para automatizar tarefas do dia a dia - e hoje estudo diariamente algo sobre tecnologia, seja uma linguagem, um framework ou algo relacionado.
