Demonstrar a estrutura de um projeto open source e práticas recomendadas para contribuição.

## 🤝 Contribuições
1. Faça um fork do projeto
2. Crie uma branch (`git checkout -b minha-branch`)
3. Realize as alterações desejadas
4. Faça um commit (`git commit -m 'Minha contribuição'`)
5. Envie para o repositório remoto (`git push origin minha-branch`)
6. Abra um Pull Request
