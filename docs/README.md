## Passo-a-passo para executar o projeto localmente

1 - Instale o Visual Studio Code (VS Code) (se ainda não tiver).

2 - Clone o repositório e abra no VS Code

3 - Abrir um CMD e digitar os comandos na ordem:<br>git clone https://github.com/renanberton/dio-lab-open-source.git
<br>cd dio-lab-open-source
<br>code .
<br>

4 - Instale a extensão Live Server

No VS Code, pressione Ctrl + Shift + X para abrir a aba de extensões.
<br>Busque por Live Server (desenvolvido por Ritwick Dey) e instale.
Execute o projeto

Abra o arquivo index.html.<br>
Clique com o botão direito e selecione "Open with Live Server".
<br>
O navegador abrirá automaticamente em localhost:5500
