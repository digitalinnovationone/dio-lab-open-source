# 🚀 Contribuição Open Source - DIO

Este repositório é uma demonstração prática do meu aprendizado sobre o fluxo de trabalho **Open Source** no GitHub, realizado através do desafio da **Digital Innovation One (DIO)**.

## 📝 Sobre o Projeto
O objetivo deste laboratório foi compreender o ciclo de vida de uma contribuição técnica, desde o processo de *Fork* e clonagem, até a criação de *Pull Requests* (PRs) e o fluxo de integração contínua.

## ✅ Minha Contribuição
Neste repositório, realizei os seguintes passos:
* **Fork:** Criação de uma cópia do projeto original para o meu perfil.
* **Documentação:** Adição do meu perfil profissional na pasta `community/`.
* **Pull Request:** Submissão da alteração para o repositório principal, respeitando as normas de contribuição.

🔗 **Confira meu perfil adicionado:** [community/rehpinheirooo.md](https://github.com/rehpinheirooo/dio-lab-open-source/blob/main/community/rehpinheirooo.md)
