## 📝 Sobre o Projeto

Este repositório foi desenvolvido como parte prática do curso **Contribuindo em um Projeto Open Source no GitHub** pela [Digital Innovation One](https://www.dio.me/). 

O grande objetivo aqui é o aprendizado prático: entender o fluxo real de uma contribuição em projetos de código aberto. Ao participar, os alunos vivenciam o ecossistema do GitHub, praticando desde o *Fork* do repositório, criação de *Branches*, até o envio e revisão de *Pull Requests*, além de explorar a estilização de perfis com a linguagem **Markdown**.
