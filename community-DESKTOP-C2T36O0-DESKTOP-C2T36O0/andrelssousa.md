## 💼 Experiências
- **Automação Industrial**: Projetos de automação de processos industriais, utilizando tecnologias como CLPs, SCADA e IHMs para otimizar a eficiência operacional.  
- **Gestão de equipe técnica**: Liderança e coordenação de equipes técnicas, garantindo a entrega de projetos dentro do prazo e com alta qualidade.  
- **Integração de sistemas**: Projetos de integração de sistemas heterogêneos, garantindo comunicação eficiente entre dispositivos e plataformas.  
- **Programação de CLPs**: Desenvolvimento e manutenção de lógicas de programação em Controladores Lógicos Programáveis (CLPs) para automação de máquinas e processos.  
- **Desenvolvimento de IHM**: Criação e implementação de interfaces homem-máquina (IHM) para operação e monitoramento de processos industriais.  
- **Scada**: Configuração e manutenção de sistemas de supervisão e aquisição de dados (SCADA) para monitoramento em tempo real.  
- **C++**: Desenvolvimento de software embarcado e aplicações de alto desempenho.  
- **Python**: Automação de tarefas, análise de dados e desenvolvimento de scripts para integração de sistemas.  
- **SQL**: Desenvolvimento e gerenciamento de bancos de dados relacionais para armazenamento e consulta de dados.  
- **VBScript**: Automação de tarefas e desenvolvimento de scripts para integração de sistemas e processos.
