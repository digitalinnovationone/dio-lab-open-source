## 🏆 Desafio

### Missão: Faça um fork deste projeto e contribua com:

- Seu perfil na pasta <code>community</code>  
- Melhorias no código  
- Correções de documentação  

> 💡 Veja exemplos na pasta [`community`](https://github.com/digitalinnovationone/dio-lab-open-source/tree/main/community)

---

## 🤝 Como Contribuir

### 📌 Fluxo Básico de Contribuição

1. **Encontre um projeto** → Explore repositórios no GitHub  
2. **Faça um fork** → Crie sua cópia do projeto  
3. **Clone o repositório** → Baixe para seu computador  
4. **Faça suas alterações** → Adicione suas melhorias  
5. **Envie um Pull Request** → Proponha suas mudanças  

<p align="center">
  <img src="https://media.giphy.com/media/3og0IPizf4zPR6VMt2/giphy.gif" width="400px" />
</p>

### ✅ 3 formas simples de ajudar:

- ✨ Adicione seu perfil na pasta `community`  
- 🐛 Corrija problemas reportados (issues)  
- 📚 Melhore a documentação do projeto
