## 💡 Desafio de Projeto
- Contribuir com **um projeto Open Source** no GitHub.  
- Enviar a **URL do seu fork** do projeto com sua contribuição.  
- Exemplo de fork: `https://github.com/falvojr/dio-lab-open-source`.

---

## 👤 Contribuindo com seu Profile README
- Crie um **Profile README** no diretório `community/` com informações sobre você.  
- Sugestões de conteúdo:
  - Badges com habilidades técnicas  
  - Cards de estatísticas do GitHub  
  - Projetos que você criou ou colaborou  
  - Links para desafios e artigos DIO  
- Inspire-se nos exemplos da pasta [`community`](https://github.com/digitalinnovationone/dio-lab-open-source/tree/main/community) e [`utils`](https://github.com/digitalinnovationone/dio-lab-open-source/tree/main/utils).

## ⭐ Outras formas de contribuir
- Resolver, responder ou indicar **issues**  
- Melhorar arquivos da pasta [`docs`](https://github.com/digitalinnovationone/dio-lab-open-source/tree/main/docs)  
- Criar ou adicionar novos **utilitários** na pasta [`utils`](https://github.com/digitalinnovationone/dio-lab-open-source/tree/main/utils)  
- Dar **star** ou fazer **fork** do repositório
