## 🏆 Desafio
<h3>Missão: Faça um fork deste projeto e contribua com:</h3>
<ul style="font-size: 1.1em;">
  <li>Seu perfil na pasta <code>community</code></li>
  <li>Melhorias no código</li>
  <li>Correções de documentação</li>
</ul>

> 💡 Dica: Veja exemplos na pasta [`community`](https://github.com/digitalinnovationone/dio-lab-open-source/tree/main/community)
