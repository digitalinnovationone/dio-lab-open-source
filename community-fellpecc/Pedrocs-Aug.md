## Projetos Destacados

* **[Projeto: Carrinho de Compras Interativo]** Descrição Geral:

Este projeto é uma simulação de um carrinho de compras com foco na interface do usuário e interatividade básica. A imagem exibe uma seção de "Carrinho de compras" onde o usuário pode selecionar produtos, definir quantidades e gerenciar itens. O design é limpo e moderno, com elementos visuais claros para facilitar a navegação.
    * [[Projeto 1](https://pedrocs-aug.github.io/Carrinho-compras/)]
* **[Projeto: Amigo-secreto]**: Descrição do Projeto: 

Aplicativo Web para Organização de Amigo Secreto

Este projeto web foi concebido para oferecer uma plataforma amigável e eficiente para a organização de eventos de "Amigo Secreto". O aplicativo apresenta uma interface limpa e moderna, permitindo aos usuários gerenciar facilmente as listas de participantes e realizar o sorteio de presentes online.
    [[Projeto 2](https://pedrocs-aug.github.io/Amigo-secreto/)]
* **[Projeto:  Plataforma de Aluguel de Board Games]**: 

Descrição do Projeto: Plataforma de Aluguel de Board Games

Visão Geral:
O projeto visa desenvolver uma plataforma online para o aluguel de jogos de tabuleiro (board games). A imagem fornecida ilustra a interface principal da plataforma, apresentando uma seleção de jogos disponíveis para aluguel, como "Monopoly", "Ticket to Ride" e "Takenoko". O foco é criar uma experiência de usuário intuitiva e eficiente para que os entusiastas de board games possam facilmente navegar, escolher e alugar seus jogos favoritos
    * [[Projeto 3](https://pedrocs-aug.github.io/Alugames/)]
