## Configuração do Ambiente de Desenvolvimento

Para começar a trabalhar neste projeto, você precisará configurar seu ambiente de desenvolvimento. Siga estas etapas:

1. **Pré-requisitos:**

   - [.NET SDK](https://dotnet.microsoft.com/download) - Instale a versão mais recente.
   - [Visual Studio](https://visualstudio.microsoft.com/) ou [Visual Studio Code](https://code.visualstudio.com/) (opcional, mas recomendado).

2. **Clone o repositório:**

   ```bash
   git clone https://github.com/seu-usuario/nome-do-repositorio.git
   ```

3. **Instale as dependências:**

   Navegue até o diretório do projeto e execute:

   ```bash
   dotnet restore
   ```

4. **Execute o projeto:**

   Use o seguinte comando para iniciar o aplicativo:

   ```bash
   dotnet run
   ```

   O aplicativo estará disponível em `http://localhost:5000` (ou outro endereço, dependendo da configuração).
