# 🚀 DIO Lab Open Source

## 📚 Sobre o Projeto

Este projeto tem como objetivo ensinar, na prática, como contribuir para projetos open source utilizando o GitHub.

Através deste laboratório, os participantes aprendem conceitos fundamentais como:

- Fork de repositórios
- Criação de branches
- Commits organizados
- Pull Requests
- Colaboração em equipe
- Uso de Markdown para documentação

O projeto serve como um ambiente seguro para iniciantes darem seus primeiros passos no mundo open source. :contentReference[oaicite:0]{index=0}
