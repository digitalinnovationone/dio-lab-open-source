# Sobre 
Formado em desenvolvimento de sistemas em ETEC e graduando em análise e desenvolvimento de sistemas na FATEC, tenho uma boa base devido ao curso técnico e estou aprofundando os conceitos no curso superior.
 # Projetos
 ## Jogo

 No trabalho de conclusão de curso do curso técnico foi desenvolvido um jogo em 2 dimensões de plataforma, fui responsável pela maior parte do código do projeto, desenvolvi algumas artes também e conteúdos da documentação.

O ambiente utilizado para o desenvolvimento do jogo foi o Game Maker Studio, que usa a linguagem GML, uma linguagem de tipagem fraca, semelhante a JavaScript. Foi a primeira vez que tive contato com essa linguagem e ambiente de desenvolvimento, foi um projeto em que pude aprender e me aprofundar mais nos conhecimentos de Git e do ambiente do Game Maker Studio.

## Forms

Foi desenvolvido um projeto CRUD para um comercio fictício utilizando Windows Forms e SQLServer, inicialmente foi utilizado um banco de dados local da rede da instituição, posteriormente o banco de dados oferecido pelo plano gratuido do site somee foi utilizado para o acesso não ficar restrito a somente a instituição.
