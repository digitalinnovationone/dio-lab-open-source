Como rodar (exemplo genérico)
1. Configurar variáveis de conexão do banco em application.properties ou .env
2. Build: mvn clean package
3. Executar: java -jar target/seu-app.jar
  - Alternativa: docker-compose up --build (para subir app + banco)
4. Dicas: não commitar credenciais, use variáveis de ambiente e ajuste pool de conexões conforme carga
