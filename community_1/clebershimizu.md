# Cleber Shimizu

I'm a software engineer working on full stack web development since 2020. My favorite stack is MERN (MongoDB, Express, React, Node), but recently I've working a lot with NextJs 14 and I'm loving it! In my recent projects, both for personal and professional cases I use the T3 stack (Next, tRPC, Prisma, Tailwind and TypeScript) whenever I can.
I like to mess around with interesting libraries like Lexical for WYSIWYG text editor and Excalidraw as a whiteboard tool.
Besides web dev, I'm deeply interested in Data Science and Machine Learning. I believe these fields are essencial for empowering business to thrive in the data flooded world.
