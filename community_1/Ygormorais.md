DIO Lab Open Source

Este repositório faz parte do bootcamp da Digital Innovation One (DIO) e tem como objetivo ensinar os fundamentos da contribuição em projetos Open Source no GitHub.

📌 Objetivo

O projeto demonstra como:

Criar um repositório no GitHub.

Fazer um fork de um repositório.

Criar branches e gerenciar versões.

Contribuir com melhorias em projetos open source.

🚀 Melhorias Implementadas

Nesta versão, foram realizadas as seguintes melhorias:

Atualização do README com informações mais detalhadas.

Melhoria na responsividade do layout.

Implementação de interatividade com JavaScript.

Refatoração do código para melhor legibilidade e desempenho.

📖 Como Usar

Faça um clone do repositório:

git clone https://github.com/Ygormorais/dio-lab-open-source.git

Acesse o diretório do projeto:

cd dio-lab-open-source

Abra o arquivo index.html no navegador.

📌 Contribuindo

Sinta-se à vontade para contribuir! Siga os passos:

Faça um fork do repositório.

Crie uma branch com a sua melhoria:

git checkout -b minha-melhoria

Faça suas alterações e commit:

git commit -m "Adicionando uma nova funcionalidade"

Envie as alterações:

git push origin minha-melhoria

Crie um Pull Request para revisão.
