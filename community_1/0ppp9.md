# Luís Magalhães

Tenho o bichinho da programação desde o BASIC e desde os meus 7 anos de idade, ou seja desde 1986.
Fiz a minha primeira página web em notepad e já com javascript para fazer uns popups. Na altura uma biblioteca bastante gira que me permitiu criar algum destaque.
Estou ligado à programação desde sempre, tendo conhecimento de linguagens antigas que agora não estão em uso como COBOL e PASCAL.
Mesmo base de dados iniciei com CLIPPER e DBASE.
Tenho explorado outras vertentes como Gestor de trafego, mas programação acaba por ser o que me move mais.
Agora quero aprofundar conhecimentos sobre IA.
