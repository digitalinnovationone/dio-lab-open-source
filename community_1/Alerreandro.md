# Guia de Contribuição

## Convenção de Commits

| Tipo de Commit | Descrição                                              | Exemplo                                 |
|----------------|--------------------------------------------------------|-----------------------------------------|
| `feat`         | Adiciona uma nova funcionalidade ao projeto.          | `feat: add login endpoint`              |
| `fix`          | Corrige um bug ou problema no projeto.                | `fix: correct password validation`      |
| `docs`         | Altera a documentação do projeto.                     | `docs: update CONTRIBUTING.md`          |
| `style`        | Mudanças de estilo que não afetam a lógica do código. | `style: format code with Prettier`      |
| `refactor`     | Refatoração de código sem alterar funcionalidade.     | `refactor: simplify UserService`        |
| `test`         | Adiciona ou modifica testes no projeto.               | `test: add unit tests for AuthService`  |
