# Projeto de Calculadora Simples em Python

## Descrição:
Este é um projeto de uma calculadora simples em Python que permite realizar operações básicas de adição, subtração, multiplicação e divisão.

## Funcionalidades:
- Realiza operações de adição, subtração, multiplicação e divisão.
- Aceita entrada de números inteiros e decimais.
- Exibe o resultado da operação.

## Como usar:
1. Baixe e instale o Python em seu computador, se ainda não o tiver.
2. Clone ou baixe este repositório em seu computador.
3. Abra o arquivo `calculadora.py` em um editor de texto ou IDE Python.
4. Execute o arquivo `calculadora.py`.
5. Siga as instruções na tela para realizar as operações desejadas.
