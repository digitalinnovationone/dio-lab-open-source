Termos

JDKs = Java Development Kits, são conjuntos de ferramentas de software usados para desenvolver aplicativos e programas na linguagem Java.

SDKs = Software Development Kit, é um conjunto de ferramentas que facilita o desenvolvimento de aplicações para uma plataforma específica, como um sistema operacional, um hardware ou um ambiente de desenvolvimento.
	Ferramentas necessárias dentro da plataforma: Compiladores, Depuradores e Bibliotecas.

APIs = Interfaces de Programação que permitem que seu aplicativo se comunique com outros sistemas ou serviços.

FRAMEWORKS = Conjunto de bibliotecas e ferramentas que fornecem uma estrutura para o desenvolvimento de aplicações.

IDE = Ambiente de Desenvolvimento Integrado, é um software que reúne diversas ferramentas essenciais e otimizando o trabalho dos desenvolvedores.
