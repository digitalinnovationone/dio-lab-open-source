Configuração das Variáveis do Sistema
	Em editar variáveis do sistema >> aba avançado >> variáveis do sistema >> na parte de usuário criará[novo] um atalho de pasta para estar sendo encurtado todo o caminho, onde seu nome sera MARVEN_HOME  e abaixo colar o caminho da pasta; 
Após dentro da PATH do usuário, acrescentar a pasta %MAVEN_HOME%\bin e repedir o mesmo procedimento na PATH do sistema.


**OBS - diferente da instalação do java que já redireciona uma pasta para guardar esse aqui você que tem que colocar onde irá querer guardar a pasta.
